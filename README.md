Django REST Framework Ads API

A RESTful API built with Django and Django REST Framework for managing user accounts and advertisements. The project provides authentication and CRUD operations for ads with a clean and scalable architecture.

⸻

🚀 Features

* User Authentication
* User Registration & Login
* Advertisement Management
* Create, Read, Update and Delete (CRUD)
* Django REST Framework API
* Static & Template Support
* Modular App Structure

⸻

🛠️ Tech Stack

* Python 3
* Django
* Django REST Framework
* SQLite (Development Database)

⸻

📁 Project Structure

Drf_project/

⸻

⚙️ Installation

Clone the repository

git clone https://github.com/YOUR_USERNAME/Drf_project.git
cd Drf_project

Create a virtual environment

python -m venv .venv

Activate the virtual environment

Windows

.venv\Scripts\activate

macOS/Linux

source .venv/bin/activate

Install dependencies

pip install -r requirements.txt

Apply migrations

python manage.py migrate

Run the development server

python manage.py runserver

The project will be available at:

http://127.0.0.1:8000/
